AI Resume-Job Description Matcher Bot

AI-powered tool for matching resumes to job descriptions with detailed analysis.

AI Resume-Job Description Matcher Bot

Tools Used: Streamlit, PyPDF, LangChain, HuggingFace Embeddings, FAISS, Google Generative AI (Gemini)

Objective: To create an interactive AI-powered application that helps job seekers analyze their resume's compatibility with a job description, providing a match score, detailed analysis, and tailored suggestions for improvement.

Highlights:

  • Developed a Streamlit web application for an intuitive user interface.
  • Implemented PDF text extraction capabilities using PyPDF to process uploaded resumes.
  • Utilized HuggingFace Embeddings (specifically "sentence-transformers/all-MiniLM-L6-v2") for converting text into numerical vectors.
  • Integrated FAISS for efficient similarity search and retrieval of the most relevant resume sections based on the job description.
  • Leveraged Google Generative AI (Gemini 2.0 Flash) through LangChain to perform a comprehensive, structured match analysis.
  • Configured the LLM to output a JSON object containing an overall match score, compatibility summary, strengths, areas for development, and tailoring suggestions.
  • Provided a clear, percentage-based match score and highlighted the most relevant resume sections with their individual relevance scores.
